- Conditional rendering (effects only show when needed)
- Efficient motion components

### Animation Timings
- **Breathing**: 3s infinite loop
- **Typing bounce**: 0.5s rapid loop
- **Sparkles**: 2.5s per particle

- **Thought bubbles**: 2s float cycle
- **Background layers**: 6-10s rotation
- **Light rays**: 3-4s pulse

### State Management
